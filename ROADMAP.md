# CSV Cleanroom Roadmap

This roadmap keeps the project focused on practical, local-first CSV cleanup.

## Near-Term

- Add more import profiles for common small-business workflows:
  - Shopify customer import
  - Airtable contacts import
  - Mailchimp audience cleanup
  - Generic CRM contact migration
- Expand CLI examples with before/after output snippets.
- Add unit coverage for profile normalization, duplicate detection, and Markdown handoff generation.
- Improve accessibility coverage for profile cards, issue cards, and export controls.
- Add sample issue reports that reviewers can inspect without running the CLI.

## Mid-Term

- Add profile packs that can be copied between projects.
- Support saved local custom profiles with export/import.
- Add a JSON schema for report output.
- Add a richer standalone HTML report generated by the CLI.
- Add documentation for integrating CSV Cleanroom in client portals or internal QA workflows.

## Non-Goals

- No hosted file uploads.
- No account system.
- No tracking or analytics.
- No committed real customer data.
- No opaque external enrichment services.
